## 🧠 Project Details

**Tester Helper** is a SA MoonLoader modification designed to simplify and improve the faction testing process.

The mod provides an interactive and easy-to-use interface where testers can manage theoretical and practical tests, select questions, view answers, and track wrong answers during a testing session.

The project focuses on a clean user interface, fast access to testing commands, and an organized testing workflow. Test questions, answers, and the time can be loaded from external **INI file**, making the mod easy to configure without modifying the main Lua script.

The goal of this project is to provide testers with a faster, cleaner, and more efficient testing experience.

---

## ✨ Features

- 🖥️ Modern and interactive mimgui interface
- 👤 Automatic player name detection based on player ID
- 📚 Theoretical and practical test modes
- 🧠 Dynamic question selector
- 💬 Quick question and answer actions
- ❌ Wrong answer tracking system (`1/3`)
- ➗ Partial answer support (`0.5/3`)
- 🔄 Wrong answer counter reset
- 📄 External INI question configuration
- 🌍 Romanian and English interface
- ⌨️ Quick menu access using a keybind (`F4`) or command (`/thinfo`)
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

This project was developed using:

- 🌙 Lua - Main scripting language and mod logic
- 🖼️ mimgui - Interactive graphical user interface
- 🎮 MoonLoader - Lua script loader for GTA: San Andreas
- 🔧 SAMPFUNCS - SA integration and game functions
- 📄 INI - Questions, answers, and time

👉 The modification was developed specifically for **SA** and requires **MoonLoader**, **mimgui**, and **SAMPFUNCS** to work properly.
